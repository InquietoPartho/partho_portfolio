Getting Started with SHAP in Python

By P.K.R. Partho • Feb 2026 • XAI Tutorial

Machine learning models, especially Deep Learning and ensemble methods like Random Forest, are often treated as "Black Boxes." In healthcare, this is unacceptable. Doctors need to know why a model predicts a high risk of hypertension.

SHAP (Shapley Additive Explanations) is a game-theoretic approach to explain the output of any machine learning model.

1. Installation

First, we need to install the library. I recommend using `pip`.

pip install shap pandas scikit-learn matplotlib

2. Loading Data and Training

For this example, we will use a standard dataset to predict diabetes progression.

import pandas as pd
import shap
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestRegressor

# Load data
X, y = shap.datasets.diabetes()
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Train a "Black Box" model
model = RandomForestRegressor()
model.fit(X_train, y_train)

3. Calculating SHAP Values

This is where the magic happens. We create an explainer object and calculate the values.

# Create the explainer
explainer = shap.TreeExplainer(model)

# Calculate SHAP values
shap_values = explainer.shap_values(X_test)

4. Visualizing Global Importance

The Summary Plot is the most powerful tool in XAI. It shows which features matter most and how they impact the output.

Figure 1: A SHAP summary plot showing feature impact on model output.
shap.summary_plot(shap_values, X_test)

Conclusion

By using SHAP, we can turn a complex Random Forest into an interpretable tool. In my research on hypertension, this allows us to tell clinicians exactly which biomarkers are driving the risk prediction.
